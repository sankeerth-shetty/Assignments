Module 3 (HTML5) Assignment

1) What are the new tags added in HTML5?

Ans.

<header>, <nav>, <section>, <article>, <aside>, <canvas>, <embed>, <footer> etc. are the new tags added in HTML5.

2) How to embed audio and video in a webpage?

Ans.

To embed audio in webpage we use:- <audio controls src="../example.mp3"></audio>




To embed video in webpage we use:- <video controls src="../example.mp4"></video>


3) Semantic element in HTML5?

Ans.

Semantic elements are those elements which describes its meaning to both developers and the browser

For example: <header>, <nav>, <section>, <article>, <aside>, <form>, <footer> etc.

4) Canvas and SVG tags

Ans.

<canvas> element is a container for graphics and is used to draw graphics, via Javascript


<svg> element is a conatiner for svg graphics and is useful for defining graphics such as circles, boxes etc.
