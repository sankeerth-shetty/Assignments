Module 1 (HTML) Assignment

1) Are the HTML tags and elements the same thing?

Ans. The difference between HTML tags and elements are that tags are in angle brackets that means tag starts with less than and ends with greater than symbol and tags are used to hold HTML elements, whereas HTML elements holds the content that means the collection of start tag, content and end tag is called HTML element.

2). What are tags and attributes in HTML?

Ans. Tags in HTML are a way of representing HTML elements whereas Attributes in HTML describes the characteristic/behaviour of HTML elements.

3). What are void elements in HTML?

Ans. Elements that have only start tag and ends without any end tag are called void elements, for example: <br>, <hr>, <input>, <img> etc. These type of elements cannot have elements as child.

4). What are HTML Entities?

Ans. In HTML there are some characters which are reserved, and these characters cannot be displayed on web page, so to use such characters HTML Entities are used to display the reserved characters.

5). What are different types of lists in HTML?

Ans. In HTML there are three types of lists - Ordered, Unordered, and Description. Ordered lists are list that starts with <ol> tag and list items will be displayed with numbers by default. Unordered lists are list that starts with <ul> tag and list items will be displayed with bullets by default. Description lists are list of terms, it starts with <dl> tag and have <dt> and <dd> tags where <dt> defines the term and <dd> describes each term.


6). What is the 'class' attribute in HTML?

Ans. Class attribute is an attribute which specifies one or more class names for an HTML element. It can be used by CSS and Javascript to do some task or to access some specific elements.

7). What is the difference between the 'id' attribute and the 'class' attribute of HTML elements?

Ans. ID attribute is unique, it is used to identify one single element whereas Class attribute can be used to identify multiple elements.

8). What are the various formatting tags in HTML?

Ans. In HTML there are various formatting tags, such as - <b> bold text, <strong> important text, <i> italic text, <em> emphasized text, <mark> marked text, <small> smaller text, <del> deleted text, <ins> inserted text, <sub> subscript text, <sup> superscript text.

9). How is Cell Padding different from Cell Spacing?

Ans. Cell padding is used to define the space between cell content and its border whereas Cell spacing is used to define the space between each cell.

10). How can we club two or more rows or columns into a single row or column in an HTML table?

Ans. In HTML to club two or more rows into a single row we use rowspan attribute whereas to club two or more columns into a single column we use colspan.

11). What is the difference between a block-level element and an inline element?

Ans. Block-level element always begins on a new line and it takes full width whereas Inline-level element takes only sufficient width required for content.

12). How to create a Hyperlink in HTML?

Ans. To create a Hyperlink in HTML page, use anchor tags <a></a>, this tag defines hyperlink, inside <a> tag href attribute is used which indicates link's destination. For example: <a href=”https://www.google.co.in/”> Google </a>


13). What is the use of an iframe tag?

Ans. In HTML iframe stands for inline frame, this tag is used to embed another HTML page or documents into the current page.


14). What is the use of a span tag? Explain with example?

Ans. Span tag is a inline element, it is used for styling purpose to the grouped inline elements, for example: <span style=”color:red”> Hello World </span>.

15). How to insert a picture into a background image of a web page?

Ans. Using background attribute we can add image as background in webpage, for example: <body background=” image path ”> content </body>

16). How are active links different from normal links?

Ans. Normal links are links on the web page that are not visited (Unvisited) or have not been clicked, whereas Active links are links that are underlined and red in colour, it is active when the link is clicked.

17). What are the different tags to separate sections of text?

Ans. In HTML there are three tags to separate sections of text - <br>, <p>, <blockquote>. <br> tag is used to break/separate the content and continue the text to the next line. <p> tag is used to define paragraph in html, it is a block level element and starts on a new line. <blockquote> tag is a block level element and it is used to display long quotation.

18). What is SVG?

Ans. SVG - Scalable Vector Graphics, it is used to define graphics for web.

19). What is difference between HTML and XHTML?

Ans. HTML stands for Hypertext Markup Language, it is a language which is used to create web pages. XHTML stands for Extensible Hypertext Markup Language, it is a standardized version of html.

20). What are logical and physical tags in HTML?

Ans. In HTML logical tags are tags which displays text according to logical styles, for example: <strong>, <ins>, <del> etc. Physical tags are tags which provide physical formatting to the text, for example: <b>, <i>, <sup>, <sub>.
