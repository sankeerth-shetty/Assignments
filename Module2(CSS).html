Module 2 (CSS) Assignment

1) What are the benefits of using CSS?

Ans.

  1. Using css saves time, as we can define a style for each element and it applies to many web pages we want as well as it also improves site speed.
  2. It is easy to make changes in website with the help of css, simply change the style in css and it will be applied to all pages that are using this style sheet.
  3. Whether it is a pc, tablet or mobile with the combination of html and css we can create responsive websites and can use them on such devices.

2) What are the disadvantages of CSS?

Ans.

  1. Due to many version of CSS, it creates confusion among web browsers.
  2. There are cross browser issues while using css.
  3. As css works differently on different browsers, so the developers need to make changes and test for compatibility, making the program run across different browsers.

3) What is the difference between CSS2 and CSS3?

Ans.

CSS2

CSS3

  • Css2 has normal styling
  • Css2 has no transitions and animations
  • Css2 has no shadows
  • Css2 has no embedded fonts
  • Css3 has better styling
  • Css3 has new transitions and animations
  • Css3 has shadows
  • Css3 has embedded fonts
  • Css3 has responsive layouts

4) Name a few CSS style components

Ans.

Selector, Property, Value – these are the components of css style. Selector - is used to select the class, id or element name. Property - is like an attribute for eg: color, background-color, font-size etc. Value - defines the value that will be assigned to property.

5) What do you understand by CSS opacity?

Ans.

In CSS, opacity is a property of an element that describes the opacity/transparency of the element.

6) How can the background color of an element be changed?

Ans.

With the help of CSS property “background-color” we can change the background of an element.
For eg:- ( background-color: blue; ) .

7) How can image repetition of the backup be controlled?

Ans.

With the help of CSS property “background-repeat” image repetition of the backup can be controlled. This property can be used to repeat background image both vertically and horizontally, and if we do not want to repeat an image we can use "no-repeat" value.

8) What is the use of the background-position property?

Ans.

In CSS “background-position” property is used to set the position of background image. The default position of background image is at the top-left corner.

9) Which property controls the image scroll in the background?

Ans.

In CSS “background-attachment” property is used to control the image scroll in the background.

10) Why should background and color be used as separate properties?

Ans.

The “background” property in CSS is a complex property, so combining it with color would make it more complex. Also background property doesn’t inherit but color does so this can make confusion.

11) How to center block elements using CSS1?

Ans.

To center block elements use “margin-left” and “margin-right” property and give value “auto” to both, use width property and give value as per required.

For example:-

                    div{
                        margin-left: auto;
                        margin-right: auto;
                        width: 15%
                    }

12) How to maintain the CSS specifications?

Ans.

The CSS specifications are maintained by the World Wide Web Consortium (W3C). Internet media type (MIME type) text/css is registered for use with CSS.

13) What are the ways to integrate CSS as a web page?

Ans.

There are 3 ways to integrate CSS as a web page :-

  1. Inline - using style attribute inside element
  2. Internal - using style element in head section
  3. External - using link to use an external css

14) What is embedded style sheets?

Ans.

Embedded style sheets refer to when we embed style sheet information into Html document using style element, it allows us to define styles for whole html document in one place.

15) What are the external style sheets?

Ans.

External style sheets are a separate CSS file that can be accessed by using link element in head section.

16) What are the advantages and disadvantages of using external style sheets?

Ans.

Advantages:-


  1. Just create one CSS file and the styles of many documents can be organised from this one single file.
  2. To make widespread changes on website design, we have to just make some single changes in stylesheet and it will be applied on all linked pages.
  3. Classes can be made for use on many html element types in many forms of the site.

Disadvantages:-


  1. In order to import style information for each file, an extra download is essential.
  2. The execution of the file may be deferred till the external style sheet is loaded.
  3. We need to test web pages with multiple browsers in order to check compatibility issues while implementing the style sheets.

17) What is the meaning of the CSS selector?

Ans.

In CSS, the Selectors are patterns used to select the html elements we want to style.


There are 9 types of selectors in CSS:-


Universal selector, element selector, id selector, class selector, descendent selector, child selector, adjacent sibling selector, general sibling selector, grouping selector.

18) What are the media types allowed by CSS?

Ans.

Given below are various media types allowed by css:-


all, aural, braille, embossed, handheld, print, projection, screen, tty, tv

19) What is the rule set?

Ans.

A rule set consists of a selector followed by a declaration block.
