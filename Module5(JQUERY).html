1) What is JQuery?

Ans.

jQuery is a JavaScript library that helps in DOM manipulation. It is fast, light-weight, feature-rich and flexible. The purpose of jQuery is to make it much easier to use JavaScript on your website. It makes things like HTML, AJAX, animation, and event handling convenient by using APIs that work across a range of browsers.

2) How to Apply CSS Using JQuery?

Ans.

css() method in jQuery is used to change the style property of the selected element.


For example:

$(“p”).css(“background-color”,”red”);
Or
$(“p”).css({“background-color”: “red”, “font-size”: “20px”});

3) How to Add Class and Remove Class in JQuery?

Ans.

to add class in jquery we use addClass() method


For example:
$(“p”).addClass(“bg-red”);


To remove class in jquery we use removeClass() method


For example:
$(“p”).removeClass(“bg-red”);

4) JQuery Animation?

Ans.

To add animation in jquery we use animate() method to create animations.


For example:
$("div").animate({ “left”: “300px”, “opacity”: “0.7”});








5) How to create slider with animation?

Ans.

To make slider in jquery we use slider()

For example:
$( "#slider" ).slider({ "value": "50", "animate":"fast", "orientation": "horizontal" });
